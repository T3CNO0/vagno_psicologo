# Website Profissional - Vagno Jonis Psicólogo 🧠✨

Este é o repositório do website institucional e profissional do **Psicólogo Vagno Jonis (CRP 03/31228)**, especializado em Terapia Cognitivo-Comportamental (TCC) e Mindfulness. O site foi desenvolvido para apresentar os seus serviços clínicos, abordagens terapêuticas, pacotes de investimento e disponibilizar materiais práticos para os pacientes.

---

## 🚀 Funcionalidades Principais

- **Apresentação Profissional & Biografia:** Secção detalhada sobre a trajetória de mais de 6 anos do psicólogo, incluindo integração de vídeo de apresentação (`sobre.mp4`).
- **Páginas de Especialidade Dedicadas:**
  - **A Clínica (`clinica.html`):** Explicação aprofundada sobre os fundamentos da Terapia Cognitivo-Comportamental baseada em evidências científicas, conceitualização cognitiva e técnicas clínicas (Reestruturação Cognitiva, Psicoeducação, Exposição Gradual, etc.).
  - **Mindfulness (`mindfulness.html`):** Espaço focado em Atenção Plena, com guias teóricos, materiais informativos integrados via Google Drive e uma galeria de vídeos com práticas guiadas no YouTube.
- **Conversão Baseada no Utilizador:** Botões de Call to Action (CTA) diretos e flutuantes para agendamento rápido de consultas via WhatsApp.
- **Tabela de Investimentos Transparente:** Listagem clara de sessões avulsas, pacotes de sessões com descontos progressivos (4, 6, 8 e 12 sessões) e Orientação Profissional.
- **Modernidade e Acessibilidade:**
  - **Dark Mode / Light Mode:** Alternador de tema nativo por JavaScript.
  - **Design Responsivo:** Adaptado perfeitamente para ecrãs móveis (Design Mobile-First) através do padrão BEM (Block-Element-Modifier).
  - **Prova Social:** Secção de depoimentos dinâmicos de pacientes.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica avançada (com tags main, section, article, nav de acordo com os padrões de acessibilidade).

CSS3: Estilização moderna recorrendo à metodologia BEM para organização de classes reutilizáveis e limpas.

JavaScript (Vanilla): Lógica simples e performática para controlo da interface do utilizador (UI), sem dependência de bibliotecas pesadas.

📝 Detalhes Técnicos Importantes
Meta Tags de Otimização: O site inclui definições estritas contra deteção automática de números de telefone (telephone=no) e tags Open Graph para SEO básico (otimização para motores de busca).

Performance: Imagens e iframes de terceiros (como os embeds do Google Drive e YouTube) utilizam o atributo loading="lazy" para garantir um carregamento inicial rápido da página.
